# Developer | 성인석

**"처음부터 끝까지 서비스를 만들어내는 개발자"**

모던 웹 기술 스택을 활용한 풀스택 개발과 클라우드 인프라 구축 경험을 바탕으로,  
안정적이고 확장 가능한 서비스를 구현하는 개발자입니다.

### 1. 설문 조사 플랫폼 (개인 프로젝트)  
> 조사원·응답자·관리자가 한 화면에서 설문을 만들고 이어서 응답할 수 있는 프로토타입

- **기간**: 2025.11 ~ 2025.11  (약 7일)
- **팀 구성**: 1명 (풀스택)  
- **내 역할**: 기획 · 백엔드 · 프런트엔드

#### 사용 기술
**Frontend**: `HTML`, `jQuery`, `Tailwind CSS`  
**Backend**: `NestJS`, `TypeScript`, `Prisma ORM`, `MSSQL`

#### 주요 기능
- 설문 생성·수정·삭제, Excel 업로드/다운로드, 분기 로직 편집
- 설문 목록 조회, 이어하기, 진행률 표시, 문항별 유효성 검증
- NestJS REST API로 설문 구조/응답 저장, CSV 내보내기

#### 내 역할
- 조사원·응답자·관리자 역할을 정의하고 정보 구조와 사용자 흐름 설계
- Prisma 스키마와 마이그레이션 자동화를 통해 타입 안정성 확보 및 DB 스키마 관리

#### 배운 점
- 토큰 기반 상태 복원으로 이어하기 기능 구현
- Prisma와 NestJS 조합으로 타입 안정성과 생산성 확보
- 선택지 기반 분기 로직 설계 및 테스트 시나리오 작성

#### 기억에 남는 이슈 해결 or 작업 경험
- **선택지 분기 표 설계**: 복잡한 설문에서 선택지 조합에 따라 문항이 건너뛰거나 반복되는 시나리오 누락 문제를 설문 흐름 표로 정리하고 조건 배열로 정의하여 해결

### 2. Mail Advisor - AI 메일 작성 어시스턴트 (개인 프로젝트)  
> 네이버 메일 작성 시 AI가 실시간으로 전문적인 메일로 재작성해주는 Chrome 확장 프로그램

- **기간**: 2025.10.16 ~ 2025.10.27
- **팀 구성**: 1명 (풀스택)  
- **내 역할**: 풀스택 개발 전담

#### 사용 기술
**Frontend**: `Chrome Extension`  
**Backend**: `NestJS`, `TypeScript`, `MySQL 8.0`, `TypeORM`, `JWT`, `bcryptjs`  
**AI/LLM**: `OpenAI API (GPT-5-nano)`, `프롬프트 엔지니어링`  
**DevOps**: `Docker`, `Docker Compose`

#### 주요 기능
- AI 기반 메일 재작성 (프롬프트 엔지니어링 활용)
- 8가지 컨텍스트 변수 설정 (직책, 업무, 톤앤매너, 목표, 수신자, 길이 정책 등)
- JWT 인증 시스템 (Access Token + Refresh Token 자동 갱신)
- OpenAI 실제 사용량 기반 토큰 차감 시스템

#### 내 역할
- 프로젝트 전체 기획 및 백엔드 개발
- OpenAI GPT 모델 활용한 정교한 프롬프트 엔지니어링 설계
- JWT 기반 인증 시스템 및 토큰 관리 로직 구현
- Docker 컨테이너 기반 백엔드 개발 환경 구축

#### 배운 점
- 프롬프트 엔지니어링을 통한 AI 출력 형식 및 품질 제어
- 실제 AI 토큰 사용량 기반 과금 로직 설계
- JWT 기반 인증 시스템 및 토큰 관리

#### 기억에 남는 이슈 해결 or 작업 경험
- **프롬프트 엔지니어링 출력 품질 제어**: 엄격한 출력 제약 규칙과 sanitize 함수를 통해 순수 본문만 반환되도록 개선
- **토큰 사용량 기반 실시간 차감**: TypeORM 트랜잭션을 활용한 동시성 제어로 토큰 잔액 업데이트 무결성 보장

#### 링크

## 기억에 남는 프로젝트
'로스트파티'는 게임 화면을 실시간으로 인식하여 파티원 정보를 자동 추출하는 시스템입니다. 개인 프로젝트로 시작했으나, 한글 및 게임 특수폰트 인식의 한계를 느껴 OCR 전문가 팀원을 모집하여 협업으로 전환했습니다.

프로젝트 기획 및 전체 시스템 구조 설계, 프론트-백 연동, API 처리를 담당했으며, 팀 리더로서 역할 분담과 주기적인 회의를 통해 목표를 공유하며 개발을 진행했습니다.

이 프로젝트를 통해 추진력과 협업을 선택할 줄 아는 유연함, 그리고 커뮤니케이션 감각을 키울 수 있었습니다.

## 다른 사람과 구별되는 나만의 강점
문제가 생겼을 때 다양한 관점에서 접근하며, 여러 해결 방안을 설계하고 최선의 선택지를 찾아 실행하는 능력을 갖추고 있습니다.

프로젝트 '로스트파티'를 진행하며 기술적·구조적 문제에 직면했을 때, 여러 대안을 나열하고 장단점을 분석한 뒤 팀원들과 함께 최적의 방법을 선택해 실행해왔습니다. 클라이언트 처리 vs 서버 처리, 직접 구현 vs 외부 API 연동 등 다양한 선택지 속에서 균형 있게 판단했습니다.

문제를 복잡하게 만들지 않으면서도 다양하게 바라보고, 해결책을 체계적으로 정리한 후 실행에 옮기는 데 강점이 있습니다.

## 성장 방향

**"기술과 비즈니스를 연결하는 개발자로 성장"**

현재 풀스택 개발 역량과 클라우드 인프라 경험을 바탕으로, 확장 가능한 시스템 아키텍처를 설계하고 구현하는 개발자로 성장하고 있습니다. 

**핵심 역량 강화 방향**
- **클라우드 아키텍처 전문성**: AWS ECS, RDS, S3 등 고급 클라우드 서비스로 확장하여 대규모 시스템 운영 경험 축적
- **시스템 안정성 및 보안**: 모니터링, 백업, 보안 체계 구축을 통한 프로덕션 환경 운영 전문성 확보
- **팀 기술 리더십**: 협업 중심의 개발 문화 구축과 기술적 의사결정을 주도하는 개발자로 성장

**목표하는 개발자상**
안정적인 시스템을 책임지면서도, 비즈니스 요구사항을 기술적으로 해결하는 문제 해결 능력을 갖춘 개발자가 되는 것이 최종 목표입니다.
